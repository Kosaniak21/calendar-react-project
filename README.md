## Build Setup:

``` bash
# Download repository:
git clone https://github.com/Kosanyak19/calendar-react-project.git 

# Go to the app:
cd react-calendar-project

# Install dependencies:
npm install

# Server with hot reload at http://localhost:3000/
npm start

# Output will be at dist/ folder
npm run build
```
